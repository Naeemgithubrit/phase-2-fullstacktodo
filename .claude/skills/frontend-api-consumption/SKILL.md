# Frontend API Consumption

## Instructions
1. Use a centralized API client.
2. Attach JWT token to every request.
3. Handle loading and error states.
4. Render only authenticated user data.
5. Follow frontend architecture conventions.

## Examples
- Calling the tasks API with an Authorization header.
- Displaying an error when the backend returns 401.
- Rendering only the current user's task list.
