# Task Domain Logic

## Instructions
1. Enforce task validation rules.
2. Ensure tasks belong to the authenticated user.
3. Control task state transitions.
4. Apply filtering and sorting rules.
5. Maintain domain consistency.

## Examples
- Validating task title length before creation.
- Toggling task completion state.
- Filtering completed versus pending tasks.
